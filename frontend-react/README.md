# SmartCampus Nexus Student App (React)

React-based student interface for SmartCampus Nexus.

## Run

```bash
npm install
npm start
```

App URL: http://localhost:3000

## Build

```bash
npm run build
```

## Environment

Create .env from .env.example and adjust if needed.

```env
REACT_APP_API_URL=http://localhost:5000/api
REACT_APP_SOCKET_URL=http://localhost:5000
```

## Included Student Features

- JWT login/register
- Student dashboard stats
- Notice board and timetable
- Attendance percentage view
- Assignment submission
- Results view
- Complaint and feedback submission
- Real-time chat
- AI chatbot query UI
- Optional face attendance upload
