# SmartCampus Nexus Admin (Angular)

Angular admin panel for SmartCampus Nexus.

## Run

```bash
npm install
npm start
```

App URL: http://localhost:4200

## Backend URL

Configured in src/environments/environment.ts:

- apiBaseUrl: http://localhost:5000/api

## Included Admin Flows

- Login/Register
- Dashboard stats
- Notice creation
- Timetable bulk generation request
- Complaint status updates
- Result publishing
